Expand description

Represents a tridiagonal matrix as 3 one-dimensional vectors.

[d0, u1,  0,   ...,       0,
 l1, d1, u2,            ...,
  0, l2, d2,
 ...           ...,  u{n-1},
  0,  ...,  l{n-1},  d{n-1},]

Fields

l: MatrixLayout

layout of raw matrix

dl: Vec<A, Global>

(n-1) sub-diagonal elements of matrix.

d: Vec<A, Global>

(n) diagonal elements of matrix.

du: Vec<A, Global>

(n-1) super-diagonal elements of matrix.
